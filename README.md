# Flicker App

Flicker is a cross-platform Flutter streaming UI concept built to run on **Android and iOS** from one codebase.

## Highlights
- Responsive layouts that adapt to different mobile screen sizes.
- Four core navigation buttons/tabs: **Home, Search, Friends, Profile**.
- OTT-inspired interactive experience with:
  - Featured hero banner
  - Trending list with **My List** toggles
  - Continue watching progress
  - Search filtering
  - Friend follow/unfollow interactions
  - Profile and account settings surface

## App identity
- App name: **Flicker**
- In-app branded icon mark ("F") with gradient identity in header.

## Run locally
1. Install Flutter SDK.
2. Run:
   ```bash
   flutter pub get
   flutter run
   ```
